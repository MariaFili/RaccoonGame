# RaccoonGame
Crossroad Phaser game. Доведи енотов в Elbrus
Проект на heroku [http://dogfinderapp.herokuapp.com/](http://dogfinderapp.herokuapp.com/)

## О приложении

Переведи енотов через дорогу, чтобы все успели на учебу к 9:00! 

## Запуск
```
npm start
```
