# RaccoonGame
Crossroad Phaser game. Доведи енотов в Elbrus

![game](https://i.imgur.com/tpDFxCr.png)

## О приложении

Переведи енотов через дорогу, чтобы все успели на учебу к 9:00! 

## Запуск
```
npm start
```
